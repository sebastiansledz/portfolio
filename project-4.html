About project

This is an implementation of eMuseumPlus 2 - a solution for clients that are already using zetcom data base application (museum+) for their art collections. I played a role of project manager, designer and REST API specialist. The client was very active during project and I conducted several workshops to reconcile client vision and technical feasibility. To support it I implemented a Figma design system that helped client to imagine the product and developers to read from client imagination.
