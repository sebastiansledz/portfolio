eMuseumPlus v2 for zetcom

Web design / development / project management

About project

For zetcom (Swiss technology company) in 2019/20 I was in charge of the project to create a new product eMuseumPlus 2 - museums and art galleries data bases online. The new solution was about to replace legacy Java application and bring modern React and Elastic stack approach. I was responsible for it from the RFP stage (to outsource part of the development) to first successful client implementations.

I managed the project from zetcom side and was a link between developers, stakeholders and clients. In later, post MVP phase, I supported sales and introduced first clients to the new ecosystem. As of now the complete product is serving such renowned clients like Louvre, Thielska Museum, Winterthur Museum, Gurlitt Museum, Canton Zurich, European Parliament and several others.

Besides project manager role I was responsible for UI/UX design of the new solution and to translate a long list of MVP requirements I created first clickable prototypes of eMuseumPlus 2. Later I have build a dedicated Figma design system to link web developers with clients vision. Thanks to this I first introduced prototyping to the company project management workflow. As a specialist I handled REST API part of the development and build complex queries to fill the new front-end with data.
