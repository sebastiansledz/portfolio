World Wide Developers

Free support, infrastructure and development to nonprofits.

I am a co-founder of group of professional volunteers. The main goal of this project is to help nonprofit organizations within system administration and web development areas, to advise and support them. We provide VPS server, maintenance, monitoring, web design and development, support and care. We also running workshops to give a guidance how to use latest technology achievements that help to fulfill organizations mission.

Doodle Awesome

Icons from Font Awesome library doodled.

I have started works on doodle version of every single icon from glorious Font Awesome library (free version). I am recreating about 1500 icons. As for a side project I have limited time to get into that so I am doodling whenever I can - every icon counts. At the end I am going to share the library as a web font and SVGs. Further in plans I have to create an app that will help works with icons set.

I am traditional man hence every icon is drawn on a paper at first. This helps me to get the feeling of shapes I need to reflect digitally.

Dummy Blog

WordPress instance for developers to test their apps.

The need for such comes up when I worked on simple Vue.js app that is fetching WordPress posts and meta via WP REST API. I just wanted to have a dummy WordPress blog to test my components and fill the layout with something real.

I found that it might be helpful for developers when they could start writing their apps without all the hassle related to running own dummy blog.

As soon as I release the beta version I will make this project open to contribute by everyone interested.
