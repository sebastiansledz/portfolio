eMuseumPlus 2 for Ithra

Web design / development / project management

About project

This was an exciting and demanding project for the King of Saudi Arabia himself. It is one of the implementation of zetcom eMuseumPlus 2 web application and Ithra stands for The King Abdulaziz Center for World Culture. As a project manager I supported sales team from the very beginning of discussions with King's representatives. One of the challenges was to manage and create RTL design and you may see results live on linked website.

I was also a sole designer of the UI/UX of this application. The entire project was implemented using Figma design system that I created for eMuseumPlus 2. I also supported developers with creating a rich API queries to fetch data from Java back-end.
