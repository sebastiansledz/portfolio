Various projects for various clients

Web design / development / project management

Here is a set of noticable projects for various clients that I was working for. Playing a role of designer, web developer or a project manager I was able to touch different stages of product development - from its earliest stage to successful implementation.

Berlinische Galerie

It was a really multidimensional project. I was taking care of the contact with client, conducting workshops, estimating works and preparing contract that covers what I was going to do.

Key task was to implement an interactive map with a rooms and available exhibitions. I created a map in Figma, exported as SVG and edited its code on HTML document directly by adding click events and CSS. The date was fetched to JSON from the back-end. Besides this there was a number of changes across the whole application to enhance it overall look and feel.

The Graphische Sammlung

During this project I took a role of project manager and developer. I was a key person to contact client, allocate tasks and deliver results. The job was about to rebuild the whole layout of the application based on design provided by client.

I managed to work on InVision sketch to translate it from graphics to live site. The works touched HTML and CSS writing, JavaScript magic and ThymeLeaf templating. Thymeleaf is a server-side Java template engine that runs legacy eMuseumPlus applications (pre eMuseunPlus 2 that I helped to build). After dozens of projects I did for clients in zetcom I found Thymeleaf syntax easy to work with and would love to touch it in future.
